## 確定した運用ルール

| ツール | 役割 | タイミング |
|--------|------|-----------|
| Google Calendar | 計画（`[BtoB]` 等） | 週初・スケ組み |
| LINE ⏱ 実動 | 実動 → `time_log` | 作業の切り替え時 |
| LINE クイズ | 習慣・歩数・できたこと等 → `log` | 朝・昼・夜（Push → そのまま回答） |
| 鬼速PDCA | 週次 Check / Adjust | **日曜**（先週の日〜土を振り返り） |

**週の区切り:** 日曜始まり・土曜終わり（PDCA `getWeekStart()` と同じ）

**5カテゴリ統一:**

| カテゴリ | カレンダー | LINE |
|----------|-----------|------|
| BtoC | `[BtoC]` | BtoC |
| BtoB | `[BtoB]` | BtoB |
| BtoG | `[BtoG]` | BtoG |
| スポット | `[Spot]` | Spot |
| レバレッジ | `[Lev]` | Lev |

**毎週水曜は休日**（`[BtoB]` 定例なしは正常。比較時に欠けとして扱わない）

**接頭辞なしの予定** — 仕事っぽい予定でも `[BtoB]` 等を付けなくてよい（5カテゴリ比較対象外）

**日曜 Plan スマホ操作** — `予算` で再開、`予算スキップ` で中断。午後クイズ完了後に Plan 未完了なら自動リマインド
