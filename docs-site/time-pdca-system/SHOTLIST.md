# スクショ撮影リスト

図解ページ `index.html` 用。撮影後、以下のファイル名で  
`docs-site/time-pdca-system/images/` に保存してください。

---

## 必須（6枚）— これが揃えばページが完成

### 01 — カレンダー（Plan）

| 項目 | 内容 |
|------|------|
| **ファイル名** | `01-calendar-week.png` |
| **アプリ** | Google Calendar（週表示） |
| **撮る画面** | 1週間分（日〜土）が見える状態 |
| **必須要素** | `[BtoB]` `[Lev]` `[Spot]` 等の接頭辞付き予定が **2種類以上** 見えること |
| **NG** | 接頭辞なしだけの週、月表示、予定がほぼ空 |
| **サイズ目安** | 横 1200px 以上（PCブラウザのスクショでOK） |

## 渡し方（Cursor）

1. 上記ファイル名のまま `images/` に置く  
2. チャットに「スクショ入れた」と送る  
3. 再デプロイまで任せる

```bash
cd docs-site/time-pdca-system
npx surge . --domain oni-pdca-time-system.surge.sh
```
